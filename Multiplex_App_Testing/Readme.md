# Testing of Multiplex Mobile Application

## Project Description
Multiplex is a mobile application that allows users to browse movies and book or purchase tickets in selected cinemas across Ukraine.

## Scope of Testing
- Mobile app flows: browsing movies, viewing cinema sessions, ticket reservation
- UI elements: tabs, buttons, menus
- Network testing using Fiddler: response simulation and error handling
- Installation, registration, and profile management

## Platform
- Mobile (Android)
- Application language: Ukrainian

## Tools Used
- Fiddler Classic v5.0.20253.3311
- Postman v11.80.1

## Environment and Devices
- Laptop with Windows 10 Pro
- Samsung Galaxy A51, Android 13

## Types of Testing
- Smoke testing
- UI testing
- Functional testing
- Integration testing
- Negative testing
- Network testing
- End-to-End testing

## Testing Focus
Testing focused on core user flows and application behavior under normal and error network conditions.
