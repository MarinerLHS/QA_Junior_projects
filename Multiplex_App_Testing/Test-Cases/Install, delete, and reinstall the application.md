# Test Case ID: TC_1
## Title: Install, delete, and reinstall the application

----

- Type of testing: Installation

- Test Object: Multiplex mobile app

- Test Type: Positive 

----

## Preconditions:
1. Mobile phone on Android platform is available and ready to use.
2. Stable connection to Wi-Fi network, internet connection is available.
3. Google Play account is available and ready to use. 

## Steps:
1. Follow the link [Multiplex](https://play.google.com/store/apps/details?id=com.interpretator.multiplex&hl=en)
2. Install apllication on your mobile phone. 
3. Check that installation was successfull, icon presents on mobile phone and it is possible to open the app. 
4. Uninstall application from mobile phone. 
5. Check that there is no app icon on your device anymore.

## Expected Result:

- Application installs and reinstalls successfully without issues


## Actual Result:

- Successfull instalation and unistalltion of application was provided.
- There was no crashes, freezes or errors during installtion or installtion.
- Procedure behavior was expected and predictable.	


- **Status**: Pass
