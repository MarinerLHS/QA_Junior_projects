# Test Case ID: TC_4
## Title: Follow an external link in the application (“Multiplex.ua”)
----

- Type of testing: Functional

- Test Object: Multiplex mobile app external links following

- Test Type: Positive 

----

## Preconditions:
1. Mobile phone on Android platform is available and ready to use.
2. Stable connection to Wi-Fi network, internet connection is available.
3. [Multiplex](https://play.google.com/store/apps/details?id=com.interpretator.multiplex&hl=en) application is installed 

## Steps:
1. Open application. 
2. Open the burger menu in the left, up corner.
3. Tap on Multiplex.ua tab.
4. Application opens browser and redirect you to Multiplex.ua site main paige.
## Expected Result:

- The Multiplex.ua website opens in the browser.

## Actual Result:

- Apllication reacts as expected.
- The external link Multiplex.ua opens in browser. Main page is shown.


- **Status**: Pass
