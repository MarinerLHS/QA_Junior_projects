# Test Case ID: TC_5
## Title: Choose a cinema theatre and view available cinema sessions
----

- Type of testing: UI/UX testing

- Test Object: Screen with sessions in selected cinema

- Test Type: Positive 

----

## Preconditions:
1. Mobile phone on Android platform is available and ready to use.
2. Stable connection to Wi-Fi network, internet connection is available.
3. [Multiplex](https://play.google.com/store/apps/details?id=com.interpretator.multiplex&hl=en) application is installed 

## Steps:
1. Open the application. 
2. Open the burger menu in the upper left corner. Select the "кінотеатри" tab.
3. In the "кінотеатри" tab, select a cinema "Victoria Gardens", Kyiv (Київ).
4. Observe the displayed screen. 

## Expected Result:

- The application displays all available sessions for the selected cinema.
- The interface is clear and understandable.
- Available films are displayed with dates, times, and cinema hall types.

## Actual Result:

- Advertisements and other unrelated information are displayed on the screen.
- Available and upcoming sessions are mixed, and it is unclear which sessions belong to the selected cinema.
- To view available dates and times, the user must select a film and tap the "Сеанси" button.

- **Status**: Fail 

## Vocabulary: 

-кінотеатри = ""cinema theaters"
