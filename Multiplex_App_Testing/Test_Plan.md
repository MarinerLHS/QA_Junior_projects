**Multiplex Application Test Plan**
## 1. Introduction
- The purpose of this test plan is to define scope, objectives, test resources for testing of Multiplex apllcication.
- The goal of this test is to ensure that main functionality of this app works correctly.
  Test covers UI, correctnes of localization and navigation to cinema seaters, instalation and deinstalation, updating. 
- Testing of network and how app reacts on different requests.
## 2. Test Objectives
- Check instalation, registration, changing info in main profile. 
- UI testing, how app reacts when you swiping and surfing the app, check how main buttons response( Filters, Overview, Burger Menu etc.) 
- Testing app for correctnes of navigation and localization, posibility to contact with cinema. 
- Folowing an external links 
- How application reacts on unexpected network response. 
## 3. Scope
**In-Scope**:
- Mobile UI and functional flows
- Network error handling
- User actions: registration, login, profile editing, ticket reservation
- Installation and uninstallation
**Out-Scope**
- Automation scripts
- Performance and load testing
